mdTEM

mdTEM EM algorithm with trimming (TEM) for data with missing values.

expand all in page

Syntax

  • out=mdTEM(Y)example
  • out=mdTEM(Y,Name,Value)example

Description

The algorithm: - At each iteration compute adjusted partial Mahalanobis distances - Rank them and set weights w_i = 1 for the lowest n*(1-alpha) rows, else 0 - Run E-step and M-step using these weights - Repeat until convergence or maxiter

example

out =mdTEM(Y)

example

out =mdTEM(Y, Name, Value)

alpha 
—proportion to trim.real number in the interval [0 0.
    

    
	
    5]
                 or empty value.
     
    
                 At each iteration compute adjusted partial Mahalanobis
                 distance and set weights w_i = 1 for the lowest
                 n*(1-alpha) rows. (e.g., 0.5 -> keep 50% with smallest
                 distances). If alpha is empty the default value which is
                 used is 0.5.
    
                 
    
	
    
       Example:  'alpha',0.1
