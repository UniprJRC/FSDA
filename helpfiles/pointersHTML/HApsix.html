HApsix

HApsix Computes psi function using Hampel proposal times x

Description

More About

Function HApsix transforms vector u as follows \[ HApsi(u) = \left\{ \begin{array}{cc} u^2 & |u| <= a \\ a \times sign(u)u & a <= |u| < b \\ a \frac{c-|u|}{c-b} \times sign(u)\times u & b <= |u| < c \\ 0 & |u| >= c \end{array} \right. \] where $a$= ctuning(2) *ctuning(1). $b$= ctuning(3) *ctuning(1). $c$= ctuning(4) *ctuning(1). The default (if input ctuning is a scalar) is $a$= 2*ctuning. $b$= 4*ctuning. $c$= 8*ctuning. It is necessary to have 0 <= $a$ <= $b$ <= $c$
