HApsider

Computes derivative of psi function using Hampel proposal

More About

Function HApsider transforms vector $u$ as follows \[ HApsi(u) = \left\{ \begin{array}{cc} 1 & |u| <= a \\ 0 & a <= |u| < b \\ -\frac{a}{c-b} & b <= |u| < c \\ 0 & |u| >= c \end{array} \right. \] where $a$= ctuning(2) *ctuning(1). $b$= ctuning(3) *ctuning(1). $c$= ctuning(4) *ctuning(1). The default (if input ctuning is a scalar) is $a$= 2*ctuning. $b$= 4*ctuning. $c$= 8*ctuning. It is necessary to have 0 <= $a$ <= $b$ <= $c$
