htmlwriteFS

htmlwriteFS Enables to create automatic HELP FILES from a specific MATLAB structure created with function mreadFS.m

Description

htmlwriteFS creates HTML files a MATLAB structure created with function mreadFS.m

More About

This function uses routine strjoin and can be used just by those which have at least MATLAB R2013a
