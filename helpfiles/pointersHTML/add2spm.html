add2spm

add2spm Adds objects (personalized clickable multilegends and text labels) to the scatter plot matrix

Description

More About

add2spm is essentially used within FSDA function spmplot.m. However its logic can be also demonstrated with MATLAB function gplotmatrix.m, as in the examples below. As default add2spm makes legends in the existing scatter plot matrix clickable or creates a clickable multilegend if the legend does not exist. Using varargin it is possible to: 1. Personalize the legend of groups in the scatterplot matrix. See option 'userleg'. 2. Add labels of the units belonging to the last data group (or to the group with the largest value in the grouping variable) of each scatter (panel). See option 'labeladd'.

References
