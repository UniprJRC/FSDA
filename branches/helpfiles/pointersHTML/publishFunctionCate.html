publishFunctionCate

Enables to create web page which contains the categorical list of functions

Description

This routins uses as input the cell which is created with routine makecontentsfileFS.m and uses template file function-cateEmpty.html which is contained inside: (main root of FSDA) filesep 'helpfiles' filesep 'FSDA; to create in a fully automatic way the categorical list of functions with automatic links for each category. The output file will be created inside (main root of FSDA) filesep 'helpfiles' filesep 'FSDA; and will have name function-cate.html The automatic help file starting from structured .m file can be created using function publishFS.m
